# amr_robot

ロボットの形状と物理寸法を一元管理する

## 設定可能パラメータ

### config/robot.yaml

| 項目 | 内容 | 影響 | 初期値 |
| --- | --- | --- | --- |
| wheel.radius | 車輪の半径 [m] | 実機と合わないと、計算上の移動速度や移動距離がずれる | 0.108 |
| wheel.track | 左右車輪の間隔 [m] | 実機と合わないと、旋回角度や旋回速度がずれる | 0.42 |
| footprint.frame | ロボット形状の基準フレーム | 障害物や空き距離を計算するときの基準になる | base_link |
| footprint.length | ロボットの前後方向の長さ [m] | 大きくすると安全側になるが、通行可能な場所が少なく判定される | 0.48 |
| footprint.width | ロボットの左右方向の幅 [m] | 大きくすると安全側になるが、狭い通路を通れないと判定しやすくなる | 0.48 |
